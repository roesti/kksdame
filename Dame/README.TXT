PROJECT TITLE: KKSDame
PURPOSE OF PROJECT: Realisierung eines Dame-Spiels (Konsole + GUI) in Java im Rahmen des berufsbezogenen Unterrichts (Informatik), Karl-Kübel-Schule, Kurs 12FO3 Jahr 2014/2015
VERSION or DATE: 0.1 / 2015
HOW TO START THIS PROJECT: main-Methode der Klasse Spiel aufrufen
AUTHORS: Robert Sattler, Yannik Fischer, Thorsten Keinz, Johannes Schneider
USER INSTRUCTIONS: 

Das Spiel orientiert sich am Brettspiel "Dame". In der CLI-Version muss jeder Zug über die Tastatur
angegeben werden. Die GUI-Version lässt sich bequem mit der Maus steuern und versucht ein realistischer
anmutendes Spielbrett darzustellen.
